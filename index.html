linux commands

Navigeren in mappen

  • pwd – toont in welke map je nu bent
  • ls – laat bestanden en mappen zien
  • cd mapnaam – ga naar een andere map
  • cd .. – ga één map terug
  • clear – maak het scherm leeg

Bestanden en mappen beheren

  • mkdir mapnaam – maak een nieuwe map
  • rmdir mapnaam – verwijder een lege map
  • rm bestandsnaam – verwijder een bestand
  • cp bron doel – kopieer een bestand
  • mv bron doel – verplaats of hernoem een bestand
  • touch bestandsnaam – maak een leeg bestand aan
  • cat bestandsnaam – toon de inhoud van een bestand
  • nano bestandsnaam – open een eenvoudige teksteditor

Informatie opvragen

  • man commando – toont de handleiding van een commando
  • history – laat je gebruikte commando’s zien
  • whoami – laat zien met welke gebruiker je bent ingelogd
  • uname -a – toont systeeminformatie
  • df -h – toont schijfgebruik
  • free -h – toont geheugenverbruik

Software en beheer

  • sudo apt update – vernieuwt de softwarelijst
  • sudo apt upgrade – installeert updates
  • sudo apt install pakket – installeert een programma
  • sudo apt remove pakket – verwijdert een programma

Netwerk en SSH

  • ping website – test internetverbinding
  • ip a – toont netwerkadressen
  • ssh gebruiker@ip – log in op een andere computer via SSH
  • exit – sluit een terminal of SSH-sessie af
